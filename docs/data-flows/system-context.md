## Technology Boundaries

| Boundary | Protocol | Authentication | Latency |
|----------|----------|---------------|---------|
| SDK вҶ’ API | HTTPS / gRPC | Bearer API key | ~1ms (local), ~10ms (remote) |
| API вҶ’ PostgreSQL | TCP (libpq) | Connection string credentials | ~1ms |
| API вҶ’ Redis | TCP (RESP) | Connection string (optional auth) | ~0.5ms |
| Dashboard вҶ’ API | HTTPS | Session cookie (NextAuth) | ~10ms |
| Webhook Worker вҶ’ External | HTTPS | HMAC-SHA256 signature | Variable |
| CLI вҶ’ API | HTTPS | Bearer API key | ~10ms |
